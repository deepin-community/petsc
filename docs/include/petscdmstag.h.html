 53: /*S
 54:   DMStagStencil - data structure representing a degree of freedom on a `DMSTAG` grid

 56:   Data structure (C struct), analogous to describing a degree of freedom associated with a `DMSTAG` object,
 57:   in terms of a global element index in each of up to three directions, a "location" as defined by `DMStagStencilLocation`,
 58:   and a component number. Primarily for use with `DMStagMatSetValuesStencil()` (compare with use of `MatStencil` with `MatSetValuesStencil()`).

 60:   Level: beginner

 62:   Note:
 63:   The component (c) field must always be set, even if there is a single component at a given location (in which case c should be set to 0).

 65: .seealso: [](ch_stag), `DMSTAG`, `DMDA`, `DMStagMatSetValuesStencil()`, `DMStagVecSetValuesStencil()`, `DMStagStencilLocation`, `DMStagSetStencilWidth()`,
 66:           `DMStagSetStencilType()`, `DMStagVecGetValuesStencil()`, `DMStagStencilLocation`
 67: S*/
 68: typedef struct {
 69:   DMStagStencilLocation loc;
 70:   PetscInt              i, j, k, c;
 71: } DMStagStencil;
