208: /*E
209:     SNESConvergedReason - reason a `SNES` method was determined to have converged or diverged

211:    Level: beginner

213:    The two most common reasons for divergence are
214: $   1) an incorrectly coded or computed Jacobian or
215: $   2) failure or lack of convergence in the linear system (in this case we recommend
216: $      testing with -pc_type lu to eliminate the linear solver as the cause of the problem).

218:    Diverged Reasons:
219: .    `SNES_DIVERGED_LOCAL_MIN` - this can only occur when using the line-search variant of `SNES`.
220:        The line search wants to minimize Q(alpha) = 1/2 || F(x + alpha s) ||^2_2  this occurs
221:        at Q'(alpha) = s^T F'(x+alpha s)^T F(x+alpha s) = 0. If s is the Newton direction - F'(x)^(-1)F(x) then
222:        you get Q'(alpha) = -F(x)^T F'(x)^(-1)^T F'(x+alpha s)F(x+alpha s); when alpha = 0
223:        Q'(0) = - ||F(x)||^2_2 which is always NEGATIVE if F'(x) is invertible. This means the Newton
224:        direction is a descent direction and the line search should succeed if alpha is small enough.

226:        If F'(x) is NOT invertible AND F'(x)^T F(x) = 0 then Q'(0) = 0 and the Newton direction
227:        is NOT a descent direction so the line search will fail. All one can do at this point
228:        is change the initial guess and try again.

230:        An alternative explanation: Newton's method can be regarded as replacing the function with
231:        its linear approximation and minimizing the 2-norm of that. That is F(x+s) approx F(x) + F'(x)s
232:        so we minimize || F(x) + F'(x) s ||^2_2; do this using Least Squares. If F'(x) is invertible then
233:        s = - F'(x)^(-1)F(x) otherwise F'(x)^T F'(x) s = -F'(x)^T F(x). If F'(x)^T F(x) is NOT zero then there
234:        exists a nontrivial (that is F'(x)s != 0) solution to the equation and this direction is
235:        s = - [F'(x)^T F'(x)]^(-1) F'(x)^T F(x) so Q'(0) = - F(x)^T F'(x) [F'(x)^T F'(x)]^(-T) F'(x)^T F(x)
236:        = - (F'(x)^T F(x)) [F'(x)^T F'(x)]^(-T) (F'(x)^T F(x)). Since we are assuming (F'(x)^T F(x)) != 0
237:        and F'(x)^T F'(x) has no negative eigenvalues Q'(0) < 0 so s is a descent direction and the line
238:        search should succeed for small enough alpha.

240:        Note that this RARELY happens in practice. Far more likely the linear system is not being solved
241:        (well enough?) or the Jacobian is wrong.

243:    `SNES_DIVERGED_MAX_IT` means that the solver reached the maximum number of iterations without satisfying any
244:    convergence criteria. `SNES_CONVERGED_ITS` means that `SNESConvergedSkip()` was chosen as the convergence test;
245:    thus the usual convergence criteria have not been checked and may or may not be satisfied.

247:    Developer Notes:
248:    This must match petsc/finclude/petscsnes.h

250:    The string versions of these are in `SNESConvergedReasons`, if you change any value here you must
251:    also adjust that array.

253: .seealso: `SNESSolve()`, `SNESGetConvergedReason()`, `KSPConvergedReason`, `SNESSetConvergenceTest()`
254: E*/
