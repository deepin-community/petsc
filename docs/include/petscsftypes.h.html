  5: /*S
  6:    PetscSF - PETSc object for setting up and managing the communication of certain entries of arrays and `Vec` between MPI ranks.

  8:    Level: intermediate

 10:   `PetscSF` uses the concept of star forests to indicate and determine the communication patterns concisely and efficiently.
 11:   A star  <https://en.wikipedia.org/wiki/Star_(graph_theory)> forest is simply a collection of trees of height 1. The leave nodes represent
 12:   "ghost locations" for the root nodes.

 14: .seealso: `PetscSFCreate()`, `VecScatter`, `VecScatterCreate()`
 15: S*/
 16: typedef struct _p_PetscSF *PetscSF;

 35: /*S
 36:    PetscSFNode - specifier of owner and index

 38:    Level: beginner

 40:   Sample Usage:
 41: .vb
 42:     PetscSFNode    *remote;
 43:     PetscCall(PetscMalloc1(nleaves,&remote));
 44:     for (i=0; i<size; i++) {
 45:       remote[i].rank = i;
 46:       remote[i].index = rank;
 47:     }
 48: .ve

 50:   Sample Fortran Usage:
 51: .vb
 52:     type(PetscSFNode) remote(6)
 53:     remote(1)%rank  = modulo(rank+size-1,size)
 54:     remote(1)%index = 1 * stride
 55: .ve

 57: .seealso: `PetscSF`, `PetscSFSetGraph()`
 58: S*/
 59: typedef struct {
 60:   PetscInt rank;  /* Rank of owner */
 61:   PetscInt index; /* Index of node on rank */
 62: } PetscSFNode;
