# TaoGetApplicationContext
Gets the user-defined context for a Tao solvers. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetApplicationContext(Tao tao, void *usrP)
```
Not Collective


## Input Parameter

- ***tao  -*** Tao context



## Output Parameter

- ***usrP -*** user context





## See Also
 `TaoSetApplicationContext()`

intermediate
