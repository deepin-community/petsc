# TaoGetMaximumFunctionEvaluations
Gets a maximum number of function evaluations. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetMaximumFunctionEvaluations(Tao tao, PetscInt *nfcn)
```
Logically Collective


## Input Parameters

- ***tao -*** the Tao solver context



## Output Parameters

- ***nfcn -*** the maximum number of function evaluations





## See Also
 `TaoSetMaximumFunctionEvaluations()`, `TaoGetMaximumIterations()`
