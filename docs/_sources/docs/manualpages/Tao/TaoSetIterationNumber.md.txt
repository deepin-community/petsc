# TaoSetIterationNumber
Sets the current iteration number. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoSetIterationNumber(Tao tao, PetscInt iter)
```
Logically Collective


## Input Parameters

- ***tao -*** Tao context
- ***iter -*** iteration number
