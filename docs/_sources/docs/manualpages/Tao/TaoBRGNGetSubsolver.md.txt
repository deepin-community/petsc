# TaoBRGNGetSubsolver
Get the pointer to the subsolver inside BRGN 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoBRGNGetSubsolver(Tao tao, Tao *subsolver)
```
Collective




## Input Parameters

- ***tao -*** the Tao solver context
- ***subsolver -*** the Tao sub-solver context
