# TaoGetType
Gets the current Tao algorithm. 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoGetType(Tao tao, TaoType *type)
```
Not Collective


## Input Parameter

- ***tao -*** the Tao solver context



## Output Parameter

- ***type -*** Tao method
