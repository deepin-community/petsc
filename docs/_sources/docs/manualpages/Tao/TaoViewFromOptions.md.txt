# TaoViewFromOptions
View a Tao options from the options database 
## Synopsis
```
#include "petsctao.h" 
PetscErrorCode TaoViewFromOptions(Tao A, PetscObject obj, const char name[])
```
Collective


## Input Parameters

- ***A -*** the  Tao context
- ***obj -*** Optional object
- ***name -*** command line option
