# PetscFunctionListDestroy
Destroys a list of registered routines. 
## Synopsis
```
#include "petscsys.h" 
PetscErrorCode PetscFunctionListDestroy(PetscFunctionList *fl)
```

## Input Parameter

- ***fl  -*** pointer to list





## See Also
 `PetscFunctionListAdd()`, `PetscFunctionList`, `PetscFunctionListClear()`
