# PetscAbsReal
Returns the absolute value of an real number 
## Synopsis
```
#include <petscmath.h>
Real abs PetscAbsReal(PetscReal v1)
```

## Input Parameter

- ***v1 -*** the double





## See Also
 `PetscMax()`, `PetscMin()`, `PetscAbsInt()`, `PetscSqr()`

beginner
