# PetscGetPetscDir
Gets the directory PETSc is installed in 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscGetPetscDir(const char *dir[])
```
Not Collective


## Output Parameter

- ***dir -*** the directory





## Fortran Note
Not for use in Fortran
