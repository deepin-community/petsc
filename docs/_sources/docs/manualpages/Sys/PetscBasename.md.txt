# PetscBasename
returns a pointer to the last entry of a / or \ separated directory path 
## Synopsis
```
#include "petscsys.h"   
const char *PetscBasename(const char a[])
```
Not Collective


## Input Parameter

- ***a -*** pointer to string





## Fortran Note
Not for use in Fortran
