# PetscTokenDestroy
Destroys a `PetscToken` 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscTokenDestroy(PetscToken *a)
```
Not Collective


## Input Parameters

- ***a -*** pointer to token





## Fortran Note
Not for use in Fortran
