# PetscStrtolower
Converts string to lower case 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscStrtolower(char a[])
```
Not Collective


## Input Parameters

- ***a -*** pointer to string





## Fortran Note
Not for use in Fortran


## See Also
 `PetscStrtoupper()`

intermediate
