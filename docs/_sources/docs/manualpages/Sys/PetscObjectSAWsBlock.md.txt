# PetscObjectSAWsBlock
Blocks the object if `PetscObjectSAWsSetBlock()` has been called 
## Synopsis
```
#include "petscsys.h"   
PetscErrorCode PetscObjectSAWsBlock(PetscObject obj)
```
Collective


## Input Parameters

- ***obj -*** the PETSc variable
