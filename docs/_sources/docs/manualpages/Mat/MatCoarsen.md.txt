# MatCoarsen
Object for managing the coarsening of a graph (symmetric matrix) 
## Synopsis
```
typedef struct _p_MatCoarsen *MatCoarsen;
```



## Notes
This is used by the `PCGAMG` to generate coarser representations of an algebraic problem
