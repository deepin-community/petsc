# MATSOLVERMATLAB
"matlab" - Providing direct solver LU for `MATSEQAIJ` matrix via the external package MATLAB. 
## Options Database Keys

- ***-pc_factor_mat_solver_type matlab -*** selects MATLAB to do the sparse factorization



## Note
You must ./configure with the options --with-matlab




## See Also
 `PCLU`, `PCFactorSetMatSolverType()`, `MatSolverType`
