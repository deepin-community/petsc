# MatBoundToCPU
query if a matrix is bound to the CPU 
## Synopsis
```
#include "petscmat.h"  
PetscErrorCode MatBoundToCPU(Mat A, PetscBool *flg)
```

## Input Parameter

- ***A -*** the matrix



## Output Parameter

- ***flg -*** the logical flag
