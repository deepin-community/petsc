# MATSEQDENSE
MATSEQDENSE = "seqdense" - A matrix type to be used for sequential dense matrices. 
## Options Database Keys

- ***-mat_type seqdense -*** sets the matrix type to `MATSEQDENSE` during a call to `MatSetFromOptions()`
