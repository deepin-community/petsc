# MatAssemblyType
Indicates if the matrix is now to be used, for example in a solver, or if you plan to continue to add or insert values to it 
## Synopsis
```
typedef enum {
  MAT_FLUSH_ASSEMBLY = 1,
  MAT_FINAL_ASSEMBLY = 0
} MatAssemblyType;
```



## See Also
 `MatAssemblyBegin()`, `MatAssemblyEnd()`

## Level
beginner
