# MATMPIDENSE
MATMPIDENSE = "mpidense" - A matrix type to be used for distributed dense matrices. 
## Options Database Keys

- ***-mat_type mpidense -*** sets the matrix type to `MATMPIDENSE` during a call to `MatSetFromOptions()`
