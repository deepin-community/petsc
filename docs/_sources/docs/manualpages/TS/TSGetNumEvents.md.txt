# TSGetNumEvents
Get the numbers of events set 
## Synopsis
```
#include "petscts.h" 
PetscErrorCode TSGetNumEvents(TS ts, PetscInt *nevents)
```
Logically Collective


## Input Parameter

- ***ts -*** the `TS` context



## Output Parameter

- ***nevents -*** number of events
