# DMSetDimension
Set the topological dimension of the `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMSetDimension(DM dm, PetscInt dim)
```
Collective on dm


## Input Parameters

- ***dm -*** The `DM`
- ***dim -*** The topological dimension





## See Also
 `DMGetDimension()`, `DMCreate()`

## Level
beginner
