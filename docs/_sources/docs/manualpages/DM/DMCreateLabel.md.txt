# DMCreateLabel
Create a label of the given name if it does not already exist in the `DM` 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMCreateLabel(DM dm, const char name[])
```
Not Collective


## Input Parameters

- ***dm   -*** The `DM` object
- ***name -*** The label name
