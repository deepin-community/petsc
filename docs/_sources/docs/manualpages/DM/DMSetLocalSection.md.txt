# DMSetLocalSection
Set the `PetscSection` encoding the local data layout for the `DM`. 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMSetLocalSection(DM dm, PetscSection section)
```

## Input Parameters

- ***dm -*** The `DM`
- ***section -*** The `PetscSection`





## Note
Any existing Section will be destroyed


## See Also
 `PetscSection`, `DMGetLocalSection()`, `DMSetGlobalSection()`

## Level
intermediate
