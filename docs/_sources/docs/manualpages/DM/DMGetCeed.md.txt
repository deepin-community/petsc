# DMGetCeed
Get the LibCEED context associated with this DM 
## Synopsis
```
#include "petscdm.h"          
PetscErrorCode DMGetCeed(DM dm, Ceed *ceed)
```
Not collective


## Input Parameter

- ***DM   -*** The DM



## Output Parameter

- ***ceed -*** The LibCEED context

intermediate
