# DMClearFields
Remove all fields from the DM 
## Synopsis
```
#include "petscdm.h"          
#include "petscdmlabel.h"     
#include "petscds.h"     
PetscErrorCode DMClearFields(DM dm)
```
Logically collective on dm


## Input Parameter

- ***dm -*** The DM
