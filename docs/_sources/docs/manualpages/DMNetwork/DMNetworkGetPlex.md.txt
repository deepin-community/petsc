# DMNetworkGetPlex
Gets the Plex DM associated with this network DM 
## Synopsis
```
#include "petscdmnetwork.h"  
PetscErrorCode DMNetworkGetPlex(DM dm, DM *plexdm)
```
Not collective


## Input Parameters

- ***dm -*** the dm object



## Output Parameters

- ***plexdm -*** the plex dm object





## See Also
 `DMNetworkCreate()`
