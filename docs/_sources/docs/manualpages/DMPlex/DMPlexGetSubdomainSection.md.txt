# DMPlexGetSubdomainSection
Returns the section associated with the subdomain 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetSubdomainSection(DM dm, PetscSection *subsection)
```
Not Collective


## Input Parameter

- ***mesh -*** The `DMPLEX`



## Output Parameters

- ***subsection -*** The subdomain section
