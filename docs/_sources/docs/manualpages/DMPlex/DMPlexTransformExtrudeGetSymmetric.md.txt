# DMPlexTransformExtrudeGetSymmetric
Get the flag to extrude symmetrically from the initial surface 
## Synopsis
```
#include "petscdmplextransform.h" 
PetscErrorCode DMPlexTransformExtrudeGetSymmetric(DMPlexTransform tr, PetscBool *symmetric)
```
Not collective


## Input Parameter

- ***tr  -*** The DMPlexTransform



## Output Parameter

- ***symmetric -*** The flag to extrude symmetrically





## See Also
 `DMPlexTransformExtrudeSetSymmetric()`

## Level
intermediate
