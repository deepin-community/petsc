# DMPlexMetricSetIsotropic
Record whether a metric is isotropic 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexMetricSetIsotropic(DM dm, PetscBool isotropic)
```

## Input parameters

- ***dm        -*** The DM
- ***isotropic -*** Is the metric isotropic?
