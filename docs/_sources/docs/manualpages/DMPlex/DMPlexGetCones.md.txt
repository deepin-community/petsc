# DMPlexGetCones
Return cone data 
## Synopsis
```
#include "petscdmplex.h"   
PetscErrorCode DMPlexGetCones(DM dm, PetscInt *cones[])
```
Not Collective


## Input Parameters

- ***dm        -*** The `DMPLEX` object



## Output Parameter

- ***cones -*** The cone for each point
