# PetscHMapISet
Set a (key,value) entry in the hash table 
## Synopsis
```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapISet(PetscHMapI ht, PetscInt key, PetscInt val)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key
- ***val -*** The value
