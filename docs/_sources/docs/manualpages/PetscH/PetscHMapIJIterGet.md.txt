# PetscHMapIJIterGet
Get the value referenced by an iterator in the hash table 
## Synopsis
```
#include <petsc/private/hashmapij.h>
PetscErrorCode PetscHMapIJIterGet(PetscHMapIJ ht, PetscHashIter iter, PetscInt *val)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator



## Output Parameter

- ***val  -*** The value
