# PetscHMapIJGetCapacity
Get the current size of the array in the hash table 
## Synopsis
```
#include <petsc/private/hashmapij.h>
PetscErrorCode PetscHMapIJGetCapacity(PetscHMapIJ ht, PetscInt *n)
```

## Input Parameter

- ***ht -*** The hash table



## Output Parameter

- ***n -*** The capacity
