# PetscHMapIDuplicate

```
#include <petsc/private/hashmapi.h>
PetscErrorCode PetscHMapIDuplicate(PetscHMapI ht, PetscHMapI *hd)
```

## Input Parameter

- ***ht -*** The source hash table



## Output Parameter
