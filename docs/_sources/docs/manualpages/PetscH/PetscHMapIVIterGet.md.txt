# PetscHMapIVIterGet
Get the value referenced by an iterator in the hash table 
## Synopsis
```
#include <petsc/private/hashmapiv.h>
PetscErrorCode PetscHMapIVIterGet(PetscHMapIV ht, PetscHashIter iter, PetscScalar *val)
```

## Input Parameters

- ***ht   -*** The hash table
- ***iter -*** The iterator



## Output Parameter

- ***val  -*** The value
