# PetscHMapObjSet
Set a (key,value) entry in the hash table 
## Synopsis
```
#include <petsc/private/hashmapobj.h>
PetscErrorCode PetscHMapObjSet(PetscHMapObj ht, PetscInt64 key, PetscObject val)
```

## Input Parameters

- ***ht  -*** The hash table
- ***key -*** The key
- ***val -*** The value
