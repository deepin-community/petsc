# VecTaggerGetType
Gets the VecTagger type name (as a string) from the VecTagger. 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecTaggerGetType(VecTagger tagger, VecTaggerType *type)
```
Not Collective


## Input Parameter

- ***tagger  -*** The Vec tagger context



## Output Parameter

- ***type -*** The VecTagger type name
