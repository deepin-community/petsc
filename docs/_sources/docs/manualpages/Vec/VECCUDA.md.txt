# VECCUDA
VECCUDA = "cuda" - A VECSEQCUDA on a single-process communicator, and VECMPICUDA otherwise. 
## Options Database Keys

- ***-vec_type cuda -*** sets the vector type to VECCUDA during a call to VecSetFromOptions()
