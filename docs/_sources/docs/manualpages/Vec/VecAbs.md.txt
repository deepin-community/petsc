# VecAbs
Replaces every element in a vector with its absolute value. 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecAbs(Vec v)
```
Logically Collective


## Input Parameters

- ***v -*** the vector





## Level
intermediate
