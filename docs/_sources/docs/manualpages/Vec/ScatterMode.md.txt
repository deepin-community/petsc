# ScatterMode
Determines the direction of a scatter 
## Synopsis
```
typedef enum {
  SCATTER_FORWARD       = 0,
  SCATTER_REVERSE       = 1,
  SCATTER_FORWARD_LOCAL = 2,
  SCATTER_REVERSE_LOCAL = 3,
  SCATTER_LOCAL         = 2
} ScatterMode;
```
