# VecLog
Replaces each component of a vector by log(x_i), the natural logarithm 
## Synopsis
```
#include "petscvec.h" 
PetscErrorCode VecLog(Vec v)
```
Not collective


## Input Parameter

- ***v -*** The vector



## Output Parameter

- ***v -*** The vector of logs





## See Also
