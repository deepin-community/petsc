# VECSEQCUDA
VECSEQCUDA = "seqcuda" - The basic sequential vector, modified to use CUDA 
## Options Database Keys

- ***-vec_type seqcuda -*** sets the vector type to VECSEQCUDA during a call to VecSetFromOptions()
