# VecNormalize
Normalizes a vector by its 2-norm. 
## Synopsis
```
#include "petscvec.h"   
PetscErrorCode VecNormalize(Vec x, PetscReal *val)
```
Collective


## Input Parameter

- ***x -*** the vector



## Output Parameter

- ***val -*** the vector norm before normalization. May be `NULL` if the value is not needed.
