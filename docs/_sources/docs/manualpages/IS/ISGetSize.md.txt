# ISGetSize
Returns the global length of an index set. 
## Synopsis
```
#include "petscis.h" 
PetscErrorCode ISGetSize(IS is, PetscInt *size)
```
Not Collective


## Input Parameter

- ***is -*** the index set



## Output Parameter

- ***size -*** the global size
