# DMLabelGetNumValues
Get the number of values that the DMLabel takes 
## Synopsis
```
#include "petscdmlabel.h"   
#include "petscsection.h"   
PetscErrorCode DMLabelGetNumValues(DMLabel label, PetscInt *numValues)
```
Not collective


## Input Parameter

- ***label -*** the DMLabel



## Output Parameter

- ***numValues -*** the number of values
