# PetscViewerGLVisType
indicates what type of GLVis viewer to use 
## Synopsis
```
typedef enum {
  PETSC_VIEWER_GLVIS_DUMP,
  PETSC_VIEWER_GLVIS_SOCKET
} PetscViewerGLVisType;
```
