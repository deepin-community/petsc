# PetscViewerPopFormat
Resets the format for a `PetscViewer`. 
## Synopsis
```
#include "petscsys.h" 
PetscErrorCode PetscViewerPopFormat(PetscViewer viewer)
```
Logically Collective


## Input Parameters

- ***viewer -*** the `PetscViewer`
