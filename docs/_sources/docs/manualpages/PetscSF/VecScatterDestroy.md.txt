# VecScatterDestroy
Destroys a scatter context created by `VecScatterCreate()` 
## Synopsis
```
#include "petscsf.h" 
PetscErrorCode VecScatterDestroy(VecScatter *sf)
```
Collective on sf


## Input Parameter

- ***sf -*** the scatter context
