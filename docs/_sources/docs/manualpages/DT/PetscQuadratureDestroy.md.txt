# PetscQuadratureDestroy
Destroys a `PetscQuadrature` object 
## Synopsis
```
#include "petscdt.h" 
PetscErrorCode PetscQuadratureDestroy(PetscQuadrature *q)
```
Collective on q


## Input Parameter

- ***q  -*** The `PetscQuadrature` object
