# PetscDSGetType
Gets the `PetscDSType` name (as a string) from the `PetscDS` 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSGetType(PetscDS prob, PetscDSType *name)
```
Not Collective


## Input Parameter

- ***prob  -*** The `PetscDS`



## Output Parameter

- ***name -*** The `PetscDSType` name





## Fortran Note
Not available from Fortran
