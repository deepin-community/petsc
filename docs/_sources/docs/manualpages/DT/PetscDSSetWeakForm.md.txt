# PetscDSSetWeakForm
Sets the weak form object 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSSetWeakForm(PetscDS ds, PetscWeakForm wf)
```
Not collective


## Input Parameters

- ***ds -*** The `PetscDS` object
- ***wf -*** The weak form object
