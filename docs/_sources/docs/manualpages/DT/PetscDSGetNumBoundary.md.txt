# PetscDSGetNumBoundary
Get the number of registered BC 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSGetNumBoundary(PetscDS ds, PetscInt *numBd)
```

## Input Parameters

- ***ds -*** The `PetscDS` object



## Output Parameters

- ***numBd -*** The number of BC
