# PetscDSView
Views a `PetscDS` 
## Synopsis
```
#include "petscds.h" 
PetscErrorCode PetscDSView(PetscDS prob, PetscViewer v)
```
Collective on prob


## Input Parameters

- ***prob -*** the `PetscDS` object to view
- ***v  -*** the viewer
