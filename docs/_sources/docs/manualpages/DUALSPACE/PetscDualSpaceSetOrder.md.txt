# PetscDualSpaceSetOrder
Set the order of the dual space 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscDualSpaceSetOrder(PetscDualSpace sp, PetscInt order)
```
Not collective


## Input Parameters

- ***sp -*** The `PetscDualSpace`
- ***order -*** The order
