# PetscFESetName
Names the FE and its subobjects 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscFESetName(PetscFE fe, const char name[])
```
Not collective


## Input Parameters

- ***fe   -*** The PetscFE
- ***name -*** The name
