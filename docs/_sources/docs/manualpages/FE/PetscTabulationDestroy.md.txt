# PetscTabulationDestroy
Frees memory from the associated tabulation. 
## Synopsis
```
#include "petscfe.h" 
PetscErrorCode PetscTabulationDestroy(PetscTabulation *T)
```
Not collective


## Input Parameter

- ***T -*** The tabulation
