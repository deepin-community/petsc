# SNESViewFromOptions
View a `SNES` based on the options database 
## Synopsis
```
#include "petscsnes.h"  
PetscErrorCode SNESViewFromOptions(SNES A, PetscObject obj, const char name[])
```
Collective


## Input Parameters

- ***A -*** the `SNES` context
- ***obj -*** Optional object
- ***name -*** command line option
