# SNES_NORM_NONE
Don't compute function and its L2 norm. 


## Notes
This is most useful for stationary solvers with a fixed number of iterations used as smoothers.


## See Also
 `SNESNormSchedule`, `SNESSetNormSchedule()`, `SNES_NORM_DEFAULT`
