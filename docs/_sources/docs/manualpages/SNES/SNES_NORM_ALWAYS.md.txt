# SNES_NORM_ALWAYS
Compute the function and its L2 norm at each iteration. 


## Notes
Most solvers will use this no matter what norm type is passed to them.


## See Also
 `SNESNormSchedule`, `SNESSetNormSchedule()`, `SNES_NORM_NONE`
