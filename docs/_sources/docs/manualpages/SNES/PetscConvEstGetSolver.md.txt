# PetscConvEstGetSolver
Gets the solver used to produce discrete solutions 
## Synopsis
```
#include "petscconvest.h" 
PetscErrorCode PetscConvEstGetSolver(PetscConvEst ce, PetscObject *solver)
```
Not collective


## Input Parameter

- ***ce     -*** The PetscConvEst object



## Output Parameter

- ***solver -*** The solver
