# PCPARMSGlobalType
Determines the global preconditioner method in `PCPARMS` 
## Synopsis
```
typedef enum {
  PC_PARMS_GLOBAL_RAS,
  PC_PARMS_GLOBAL_SCHUR,
  PC_PARMS_GLOBAL_BJ
} PCPARMSGlobalType;
```
