# PCSide
If the preconditioner is to be applied to the left, right or symmetrically around the operator. 
## Synopsis
```
typedef enum {
  PC_SIDE_DEFAULT = -1,
  PC_LEFT,
  PC_RIGHT,
  PC_SYMMETRIC
} PCSide;
```
