# PCAmgXGetResources
get AMGx's internal resource object 
## Synopsis
```
#include "petscpc.h" 
PETSC_EXTERN PetscErrorCode PCAmgXGetResources(PC pc, void *rsrc_out)
```
Not Collective


## Input Parameter

- ***pc -*** the PC



## Output Parameter

- ***rsrc_out -*** pointer to the AMGx resource object
