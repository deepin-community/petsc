# PetscSectionSetUp
Calculate offsets based upon the number of degrees of freedom for each point. 
## Synopsis
```
#include "petscsection.h"   
PetscErrorCode PetscSectionSetUp(PetscSection s)
```
Not Collective


## Input Parameter

- ***s -*** the `PetscSection`
